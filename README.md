# SmartWallet

Implementación de una clase `SmartWallet` en Java con pruebas unitarias usando JUnit 5 y Maven.

## Autor
Jimmy Arias

## Descripción
Este proyecto implementa la lógica de una billetera virtual (`SmartWallet`) que permite realizar depósitos y retiros, con reglas de negocio específicas como límites de saldo según el tipo de usuario.

## Cómo ejecutar las pruebas
Es necesario tener Maven instalado. Luego en la raíz del proyecto, de deberá ejecutar el siguiente comando:

```bash
mvn test
```

Y asi se compilaran todos los test de `SmartWalletTest`.
